# Gato-Block
A cat game to avoid obstacles. 

https://scratch.mit.edu/projects/1274045211/

Avoid obstacles and jump over pits! To move, use the up and down arrow, or the w and s keys. To jump, click space. Collect as much food as you can!

Song: Nyan Cat - Digidudes
